# Clustering Keywords in Environmental Reports

## Overview
This project focuses on analyzing environmental reports by extracting and clustering important keywords. 
It helps organize large text documents by grouping related environmental terms, making reports easier to understand and analyze.

## Objective
- To identify key themes from environmental reports
- To group similar keywords using clustering techniques
- To support better analysis of environmental data

## Features
- Text preprocessing (cleaning and formatting)
- Keyword extraction from environmental reports
- Clustering of related keywords
- Identification of major environmental themes

## Technologies Used
- Python
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- K-Means Clustering

## Applications
- Environmental report analysis
- Topic identification
- Research support
- Data organization
